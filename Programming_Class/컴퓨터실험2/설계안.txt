202202177 차태현

1. 개발 프로그램 사용자
	: 짬뽕 & 직원

2. 개발 용도
- 매장 기타 재고 관리
	: 기타 입고 관리
	: 기타 출고 관리
	: 기타 재고 확인
	

3. 작업 순서
3-1. 입고 관리
	: 사용자가 메인메뉴 > 입고 관리 선택
	: 사용자가 모델명 입력창에 기타 모델명 입력
	: 프로그램은 기존에 보유하고 있는 데이터가 있으면 나머지 항목 자동 입력
  : 항목은 종류(일렉, 통, 베이스, 클래식, 우쿨렐레)와 제조사, 가격, 줄 갯수를 제공함
	: 사용자가 웹 검색 버튼을 누를 시 프로그램은 웹에서 정보를 받아옴
  : 사용자가 최종적으로 정보를 확인함
  : 사용자가 입고 처리 버튼을 누름
  : 프로그램이 DB에 데이터를 저장함

3-2. 출고 관리
	: 사용자가 메인메뉴 > 출고 관리 선택
	: 사용자가 모델명 입력창에 기타 모델명 입력
	: 프로그램은 모델명에 따른 현재 재고 현황을 표시함
	: 사용자가 판매 고객 정보를 입력함
  : 사용자가 출고 처리 버튼을 누름
  : 프로그램이 DB에 수정된 데이터를 저장함

3-3. 기타 재고 확인
	: 사용자가 메인메뉴 > 검색 선택
  : 사용자가 조건을 입력하고 검색 버튼을 누름
  : 조건은 종류(일렉, 통, 베이스, 클래식, 우쿨렐레)와 제조사, 가격대, 줄 갯수를 제공함
	: 프로그램은 저장돼있는 기타들의 목록과 입력받은 조건을 비교함
	: 프로그램은 조건에 부합하는 기타가 존재할 경우 재고 현황과 함께 결과 목록을 출력함
  : 사용자가 우측 상단의 정렬 기준(내림차순, 오름차순, 가격 낮은순, 가격 높은순)을 누      르면 해당 기준으로 결과 페이지가 정렬됨.
